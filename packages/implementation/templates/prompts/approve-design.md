# Specta approve-design workflow

Approve only the latest reviewed Technical Design revision.

Before invoking the CLI helper:

- Present the target project and detected or proposed framework profile to the developer.
- Confirm that a blank-project framework choice is intentional.
- Confirm that the design contains declarations and structure only.
- Do not approve automatic framework migrations for an existing project.

Specta validates the TypeScript language boundary and resolves all cross-Epic dependencies.
