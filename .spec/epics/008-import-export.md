# Epic 008 — Import & Export

## Goal

Allow Specta to integrate with existing planning systems.

## Description

Import external specifications and export Specta artifacts. Imported artifacts
immediately become part of the Workspace Graph and are available to future
workflows.

## User Stories

- Import Spec Kit.
- Import Markdown.
- Export graph.
- Export specifications.

## Deliverables

- `specta import`
- `specta export`

## Acceptance Criteria

- Imports succeed.
- Graph populated.
- Exports generated.
- Imported artifacts are available to future workflows.
