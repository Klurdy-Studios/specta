# Epic 004 — Specification & Code Analysis

## Goal

Understand both project specifications and implementation.

## Description

Specta parses markdown specifications and source code into a common graph representation.

## User Stories

- Parse markdown.
- Parse TypeScript.
- Detect symbols.
- Detect dependencies.
- Detect tests.

## Deliverables

- Markdown parser
- TypeScript parser

## Acceptance Criteria

- Specifications parsed.
- Source code parsed.
- Graph nodes created.
