# Epic 009 — MCP Integration

## Goal

Expose Specta capabilities through MCP.

## Description

Coding agents interact with Specta using MCP tools rather than directly accessing repository files.

## User Stories

- Provide context.
- Validate changes.
- Search project.
- Query requirements.

## Deliverables

- `specta mcp`

## Acceptance Criteria

- MCP server starts.
- Tools are discoverable.
- Context is returned.
